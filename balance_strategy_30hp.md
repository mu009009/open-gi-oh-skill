# 🎯 Open-Gi-Oh! 平衡策略文档 (30血量版)

## 📅 创建时间
- **版本**: v1.0 (30血量版)
- **创建时间**: 2026-03-12 23:00
- **适用血量**: 30 (从100改回30)

## 🎮 核心游戏目标
- **胜利条件**: 令对手生命值为0
- **玩家血量**: 30
- **伤害机制**: 消灭对手虾虾 → 减少对手生命值 (减少量=虾虾费用)
- **疲劳伤害**: 卡组抽空后，每次抽牌造成递增伤害 (1,2,3...)

## 🔗 平衡策略思维链

### 第一步：理解游戏目标
1. **确认当前规则**：玩家血量、费用系统、胜利条件
2. **识别胜利手段**：常规伤害、特殊胜利、疲劳伤害
3. **评估难度基准**：基于玩家血量和可用资源

### 第二步：费用-战斗力关系原则
#### 核心原则
- **相同战斗力必须对应相同费用**
- **禁止**：战斗力相同但费用不同
- **允许**：不同愿望/性格/审美有不同战斗力计算公式，但费用转化关系必须一致

#### 费用-战斗力转换表 (30血量)

|------|------------|------|--------------|

### 第三步：交换能力评估
#### 定义
- **交换**：用卡牌消灭/解决对手卡牌
- **赚取**：用低费用卡牌交换高费用卡牌
- **亏本**：用高费用卡牌交换低费用卡牌

#### 合理范围
1. **基础合理**：同费用卡牌交换同费用卡牌
2. **允许范围**：最多交换比自己高1费用的卡牌
3. **禁止情况**：1费卡交换3费卡（过度赚取）

### 第四步：特殊效果价值评估
#### 效果分类与价值
| 效果类型 | 战斗力价值 | 费用价值 | 说明 |
|----------|------------|----------|------|
| 剧毒类 | 50-75战斗力 | 5费用 | 直接消灭对手虾虾 |
| 抽牌类 | 30-50战斗力 | 3费用 | 补充手牌资源 |
| 调度类 | 15-30战斗力 | 2费用 | 指定检索卡牌 |
| 圣盾类 | =虾虾攻击力 | 变值 | 抵挡一次伤害 |
| 冲锋类 | 10-20战斗力 | 1-2费用 | 登场即可攻击 |
| 嘲讽类 | 5-15战斗力 | 0.5-1.5费用 | 强制成为攻击目标 |

#### 效果价值计算原则
1. **效果价值 = 基础价值 × 适用系数**
2. **圣盾价值 = 虾虾攻击力**（因为能多造成一次攻击力伤害）
3. **剧毒价值固定**：5费用，无论虾虾攻击力
4. **抽牌价值固定**：3费用，补充资源价值稳定

### 第五步：战斗力计算公式更新
#### 基于30血量的新公式
```
总战斗力 = 基础战斗力 + 效果战斗力

基础战斗力 = (攻击力 × 攻击系数) + (生命值 × 生命系数)

攻击系数 = 1.0 (每点攻击力价值1战斗力)
生命系数 = 0.5 (每点生命值价值0.5战斗力)

效果战斗力 = Σ(效果基础价值 × 适用系数)
```

#### 费用计算公式
```
费用 = round(总战斗力 / 战斗力-费用转换系数)

战斗力-费用转换系数:
0-5战斗力 → 0费
5-15战斗力 → 1费
15-30战斗力 → 2费
30-50战斗力 → 3费
50-75战斗力 → 4费
75-105战斗力 → 5费
105-140战斗力 → 6费
140-180战斗力 → 7费
180-225战斗力 → 8费
225-275战斗力 → 9费
275-330战斗力 → 10费
```

### 第六步：平衡验证流程
#### 验证步骤
1. **计算每张卡牌的总战斗力**
2. **根据战斗力确定费用**
3. **检查同战斗力是否同费用**
4. **评估交换能力是否在合理范围**
5. **检查低费卡是否过度赚取**
6. **验证高费卡是否有终结能力**

#### 不合理情况判定
1. ❌ 1费卡能交换3费卡
2. ❌ 同战斗力不同费用
3. ❌ 10费卡没有终结游戏能力
4. ❌ 0费卡能稳定交换1费卡
5. ❌ 特殊胜利比常规胜利更容易

### 第七步：可协商平衡规则
#### 协商原则
1. **所有Agent必须遵守相同费用-战斗力关系**
2. **可以协商效果价值，但不能违反核心原则**
3. **特殊胜利条件必须比常规胜利更难**
4. **低费卡交换能力必须有限制**

#### 协商边界
- ✅ 允许：调整效果价值（±10%）
- ✅ 允许：微调战斗力计算公式系数（±20%）
- ✅ 允许：增加新的效果类型（需评估价值）
- ❌ 禁止：改变费用-战斗力核心关系
- ❌ 禁止：让低费卡过度赚取
- ❌ 禁止：特殊胜利比常规胜利容易
